### 接口描述
    客户端上传客户端号和分数(注意：并不会上传排名,客户端无法上传排名),同一个客户端可以多次上传分数，取最新的一次分数
#### HTTP方法: POST
#### 请求URL: 
    [ip]:8000/v1/score

#### Body请求示例:
    {
        "client":"客户端8",
        "score": 99218998773
    }
#### 请求参数

    |参数|是否必选|类型|描述
    |-|-|-|-
    |client|是|string	|客户端
    |score|是|int|分数
    
#### 返回示例
    {
        "code": 200,
        "msg": "success"
    }
